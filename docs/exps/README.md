# Experiment notes

Scratch records for individual experiments — tuning sweeps, profiler dumps, dead ends, negative
results. One file per experiment: `docs/exps/<short-name>.md`.

Keep the curated, main-line story in `docs/optimization.md`; only its landed steps matter there.
This directory is the lab notebook: messy, detailed, and fine to leave half-finished. When an
experiment produces a landed change, summarize it as one entry in `docs/optimization.md` and link
back here if the detail is worth keeping.
